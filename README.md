## MySQL

版本：采用最新版本 8.0

解释：如果你采用低版本，可能有些功能无法使用（例如窗口函数）

```text
mysql> select version();
+-----------+
| version() |
+-----------+
| 8.0.22    |
+-----------+
1 row in set (0.00 sec)
```

## Leetcode 数据库刷题感悟

（1）有时候题目描述的 **需求并不是很清楚**，这个时候需要  **先做，在做的过程中不断明确需求。**

（2）有时候 **情况特别复杂，就从最简单的情况开始做**，例如找出 薪水第N高的用户，那么先想找出薪水第一高的用户，然后第二高的用户，最后考虑第N高的用户
